# Pior Filme do Golden Raspberry Awards

Essa é uma aplicação para exibir dados da API de Pior Filme do Golden Raspberry Awards
Foi desenvolvida utilizando as seguintes tecnologias:

- Next.JS
- TypeScript
- axios

## Instalação

```bash
git clone git@github.com:renato-aguena/outsera-front-end.git
cd outsera-front-end
npm install
```

## Comandos

Para executar o projeto em modo de desenvolvimento:
```bash
npm run dev
```

Para montar o pacote para produção executar comando:
```bash
npm run build
```
